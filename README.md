# 巴哈姆特 職場甘苦談 文本分析

# 前言
### 為甚麼要選巴哈姆特 - 職場版
新冠病毒(COVID-19)來了，改變了我們的生活，也影響了職場。於是我看向巴哈姆特-職場甘苦談，這裡滿滿的都是各種職場的故事，其中不乏疫情下的工作變化與心路歷程。我將進行文本分析，希望能對新冠病毒如何改變職場有更深入的理解，也希望我的研究能為即將面臨職場的同學提供一些實用的參考。
